KOSTADINOVIC NEMANJA

CNN en C apprentissage sur les données MNIST (Yann LeCun THE MNIST DATABASE of handwritten digits) 


Pour compiler : main 

Pour lancer le modèle avec la phase d'apprentissage puis test : ./main train_test, le modèle apprend sur l'ensemble des 60000 exemples et teste 10000 exemples

Pour lancer seulement la phase de test: ./main test, le modèle teste les 10000 exemples.

La précision obtenue en moyenne est de 0.97!
